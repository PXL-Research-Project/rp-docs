## Crash Course Computer Science

This selection from **Crash Course Computer Science** provides a comprehensive overview of how computers evolved from simple mechanical devices to the complex virtualized systems we use today. The series moves through the "ladder of abstraction," beginning with the physics of transistors and building up to the high-level software and global networks that define modern life.

### Early Computing

[https://www.youtube.com/watch?v=O5nskjZ_GoI](https://www.youtube.com/watch?v=O5nskjZ_GoI)

This video explores the origins of computation, starting with the abacus in 2500 BC and moving through mechanical calculators like the Step Reckoner. It highlights how the need for automated calculation led to early general-purpose designs like Charles Babbage's Analytical Engine.

### Electronic Computing

[https://www.youtube.com/watch?v=LN0ucKNX0hc](https://www.youtube.com/watch?v=LN0ucKNX0hc)

The transition from mechanical gears to electronic switches is detailed here, covering relays, vacuum tubes, and the invention of the transistor. It explains how these components allowed for room-sized behemoths like ENIAC to eventually shrink into the microchips used today.

### Boolean Logic & Logic Gates

[https://www.youtube.com/watch?v=gI-qXk7XojA](https://www.youtube.com/watch?v=gI-qXk7XojA)

This episode explains how computers use binary to represent true and false values through Boolean algebra. It demonstrates how transistors are arranged to create fundamental logic gates like NOT, AND, and OR, forming the basis of all digital computation.

### Representing Numbers and Letters with Binary

[https://www.youtube.com/watch?v=1GSjbWt0c9M](https://www.youtube.com/watch?v=1GSjbWt0c9M)

Building on logic gates, this video shows how complex data like integers, floating-point numbers, and text are stored as sequences of ones and zeros. It covers the evolution of character encoding from the 7-bit ASCII standard to the global reach of Unicode.

### How Computers Calculate - the ALU

[https://www.youtube.com/watch?v=1I5ZMmrOfnA](https://www.youtube.com/watch?v=1I5ZMmrOfnA)

The Arithmetic and Logic Unit (ALU) is introduced as the mathematical brain of the computer. By combining logic gates into circuits like the Ripple Carry Adder, the CPU gains the ability to perform the arithmetic operations that underpin software.

### Registers and RAM

[https://www.youtube.com/watch?v=fpnE6UAfbtU](https://www.youtube.com/watch?v=fpnE6UAfbtU)

This video explains how computers remember information using gated latches and registers to store individual bits. It scales these concepts up to create Random Access Memory (RAM), allowing the system to quickly read and write data at specific addresses.

### The Central Processing Unit (CPU)

[https://www.youtube.com/watch?v=FZGugFqdr60](https://www.youtube.com/watch?v=FZGugFqdr60)

The CPU is brought to life by combining the ALU, registers, and a control unit to execute the fetch-decode-execute cycle. This episode illustrates how the clock keeps every operation in lockstep, enabling the processor to run programs at gigahertz speeds.

### Instructions & Programs

[https://www.youtube.com/watch?v=zltgXvg6r3k](https://www.youtube.com/watch?v=zltgXvg6r3k)

Software is introduced as a series of instructions that tell the hardware what to do, using opcodes and memory addresses. It explains how conditional jumps allow for loops and decision-making, giving the computer its programmable power.

### Advanced CPU Designs

[https://www.youtube.com/watch?v=rtAlC5J1U40](https://www.youtube.com/watch?v=rtAlC5J1U40)

To overcome the bottleneck of slow RAM, modern CPUs utilize advanced techniques like instruction pipelining, caching, and speculative execution. These optimizations allow processors to execute multiple instructions per clock cycle and manage data more efficiently.

### Integrated Circuits & Moore's Law

[https://www.youtube.com/watch?v=6-tKOHICqrI](https://www.youtube.com/watch?v=6-tKOHICqrI)

The move from discrete components to integrated circuits solved the "tyranny of numbers," allowing millions of transistors to be etched onto silicon wafers. Moore's Law describes the exponential trend of this miniaturization, which has fueled the computing revolution for decades.

### Memory & Storage

[https://www.youtube.com/watch?v=TQCr9RV7twk](https://www.youtube.com/watch?v=TQCr9RV7twk)

This video distinguishes between volatile RAM and non-volatile storage like hard drives and magnetic tape. It explores the memory hierarchy, where different technologies are balanced to provide a mix of speed, capacity, and cost-effectiveness.

### The Personal Computer Revolution

[https://www.youtube.com/watch?v=M5BZou6C01w](https://www.youtube.com/watch?v=M5BZou6C01w)

The 1970s saw the birth of microcomputers like the Altair 8800 and the Apple II, making computing accessible to individuals for the first time. The arrival of the IBM PC established the open architecture and software standards that still dominate the industry.

### Graphical User Interfaces

[https://www.youtube.com/watch?v=XIGSJshYb90](https://www.youtube.com/watch?v=XIGSJshYb90)

Moving away from text-based command lines, the GUI introduced windows, icons, and mouse-driven interaction. Originating at Xerox PARC and popularized by the Apple Macintosh, this metaphor made computers intuitive enough for the general public.

### Screens & 2D Graphics

[https://www.youtube.com/watch?v=7Jr0SFMQ4Rs](https://www.youtube.com/watch?v=7Jr0SFMQ4Rs)

Early displays evolved from simple vector scans to raster-based bitmap screens where every pixel is mapped in memory. This video explains how the frame buffer allows programs to render shapes and text by manipulating pixel intensities.

### 3D Graphics

[https://www.youtube.com/watch?v=TEAtmCYYKZA](https://www.youtube.com/watch?v=TEAtmCYYKZA)

Rendering a 3D world onto a 2D screen requires complex math for projection, scanline rendering, and shading. The Graphics Processing Unit (GPU) was developed to handle these massive parallel calculations, enabling the realistic visuals found in modern gaming.
