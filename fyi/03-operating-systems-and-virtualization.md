## Crash Course Computer Science

### Operating Systems

[https://www.youtube.com/watch?v=26QPDBe-NB8](https://www.youtube.com/watch?v=26QPDBe-NB8)

Operating systems act as an intermediary between hardware and software, managing resources through multitasking and virtual memory. From the early batch systems to modern kernels like Linux and Windows, the OS provides a stable environment for applications to run.

### Files & File Systems

[https://www.youtube.com/watch?v=KN8YgJnShPM](https://www.youtube.com/watch?v=KN8YgJnShPM)

This video explains how data is organized into files and managed by hierarchical file systems. It covers how file formats use headers and metadata to interpret raw binary, and how directories keep thousands of files organized into logical directorys.
