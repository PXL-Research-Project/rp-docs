## Crash Course Computer Science

### Computer Networks

[https://www.youtube.com/watch?v=3QhU9jd03a0](https://www.youtube.com/watch?v=3QhU9jd03a0)

The basics of networking are explored through Local Area Networks (LANs) and the Ethernet protocol. It describes how packet switching and routing allow data to travel across a decentralized network by breaking information into small, manageable chunks.

### The Internet

[https://www.youtube.com/watch?v=AEaKrq3SpW8](https://www.youtube.com/watch?v=AEaKrq3SpW8)

This episode dives into the protocols that power the global internet, specifically the TCP/IP suite. It explains how IP handles addressing and routing, while TCP ensures reliable delivery, and DNS maps human-readable domain names to numeric IP addresses.

### The World Wide Web

[https://www.youtube.com/watch?v=guvsH5OFizE](https://www.youtube.com/watch?v=guvsH5OFizE)

The World Wide Web is an application that runs on top of the internet, using HTML for content and HTTP for transfer. It traces the web's history from Tim Berners-Lee's first browser to the rise of search engines and the modern debate over net neutrality.
